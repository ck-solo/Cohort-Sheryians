1. Selecting of Element
-------------------------
Selecting of element is the first step in DOM manipulation
There are two ways to select element
1. getElementById
2. getElementsByClassName


2. Changing HTML
-----------------
changuing HTML is the second step in DOM manipulation
innerHTML is used to change the HTML

3. Changing CSS
---------------
Changing CSS is the third step in DOM manipulation
style is used to change the CSS
heading.styke.color = "red"

4. EventListener    
---------------
EventListener is the fourth step in DOM manipulation
